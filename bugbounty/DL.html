Directory Listing

Summary: Directory Listing is a vulnerability in which the server has misconfiguration and exposes the internal and hidden directories from the server to the public internet.
Sometimes an attacker can gain sensitive information from the server like backup files, source code, API keys, password files etc.


Severity:   Medium  

Complexity: Easy 

From: Remote / External

Steps to Reproduce:

Proof of Concept: Attached in the Video


Impact: An Adversary can carry out Directory Listing to gain sensitive information from the target server
