Authentication Bypass 

Summary: Authentication Bypass is a dangerous vulnerability that is found in Web Applications. Attackers can bypass the control mechanisms that are used by the underlying web application like OTP, Captcha, 2FA, Email verification etc. 
An Attacker can perform a  complete Account takeover of a Victim.


Severity:   High  

Complexity: Easy 

From: Remote / External
